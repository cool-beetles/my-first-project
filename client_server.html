Client-server

The whole story begins when you want to send some content.

What is it all about?

The base of internet application is the client-server architecture. In this type of architecture client sends request for some particular service to server. Server perform some computing and returns appropriate response to client. Devices are connected via Internet- there could be multiple clients and servers instances, however as a rule, each device could play only one of these roles.

In case of web application- internet browser is the client- it displays user interface, collect data from user inputs, performs a basic editing and send it to the server. Web server realizes clients request and sends html code, which is rendered by browser in order to display message to user.
